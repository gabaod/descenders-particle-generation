# descenders-particle-generation
54 various particle systems to easily generate<br><br>
put .shader files into Shaders<br>
put .cs file into Assets/Editor<br>
Create a new folder Assets/Particle_Materials<br>
Create 6 new materials using same name as the shader names assigning one shader to each material<br><br>
goto tools Particle System Generation and select object it should be centered on and which one you want to be created and hit Create<br>
hit generate all to see all of them
